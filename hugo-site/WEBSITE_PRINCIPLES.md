# Website principles

- Hugo generates a static, accessible site suitable for GitHub Pages.
- All internal asset URLs are subpath-safe for `/nikolauswolf/`.
- The design uses a warm, restrained palette with Humboldt red as a small accent.
- The homepage introduces Nikolaus briefly; the biography provides a distinct career narrative.
- Writings render server-side, with optional client-side filtering and sorting.
- Publication folders use permanent slugs and Markdown so future updates do not require editing templates.
- The footer credit is disabled and the requested invisible discovery metadata is enabled.
- GitHub Actions builds Hugo first and Pagefind second.
