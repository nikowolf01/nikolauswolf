## Rosés–Wolf database on regional GDP

**Version 7, 2025**

The Rosés–Wolf database provides data on the economic development of European NUTS-2 regions for 1900–2022. It includes nominal GDP in 1990 and 2011 purchasing-power parity, population, area, and sector-level employment shares.

[Access the data, downloads, and documentation at CEPR](https://cepr.org/node/424487)

When using the data, please cite: **Rosés–Wolf database, version 7 (2025).**
